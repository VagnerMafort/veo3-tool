Política de Privacidade — Kaelum Studio

Última atualização: Abril 2026

1. Dados Coletados

Coletamos apenas os dados necessários para o funcionamento da plataforma:

  • Nome e email (cadastro)
  • Dados de pagamento (processados pela Stripe, não armazenamos cartões)
  • Conteúdo gerado (roteiros, imagens, vídeos)
  • Chaves de API fornecidas pelo usuário (armazenadas de forma segura)

2. Uso dos Dados

Seus dados são utilizados exclusivamente para:

  • Fornecer o serviço de geração de vídeos
  • Processar pagamentos
  • Comunicações sobre sua conta e o serviço

3. Compartilhamento

Não vendemos ou compartilhamos seus dados pessoais com terceiros, exceto:

  • Stripe (processamento de pagamentos)
  • OpenAI e MiniMax (geração de conteúdo — apenas os prompts são enviados)

4. Segurança

Utilizamos criptografia SSL/TLS para proteger a comunicação. Senhas são armazenadas com hash seguro. Chaves de API são armazenadas de forma protegida no servidor.

5. Seus Direitos

Você pode a qualquer momento:

  • Acessar seus dados pessoais
  • Solicitar correção ou exclusão dos seus dados
  • Cancelar sua conta

6. Cookies

Utilizamos cookies de sessão para manter você logado. Não utilizamos cookies de rastreamento ou publicidade.

7. Contato

Para questões sobre privacidade: suporte@grupomafort.com
